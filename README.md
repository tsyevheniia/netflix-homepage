1. Clone the repository:
    ```sh
    git clone https://github.com/tsyevheniia/netflix_homepage.git
    ```
2. Navigate to the project directory:
    ```sh
    cd your-repo
    ```
3. Install dependencies:
    ```sh
    npm install
    ```
